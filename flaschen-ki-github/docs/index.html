Funktionen

Barcode

Flaschennummern direkt mit der Kamera erfassen.

Offline

Daten lokal auf dem Android-Handy speichern.

GPS & Zeit

Standort, Datum und Uhrzeit automatisch dokumentieren.

Excel-Export

CSV-Datei teilen und direkt in Excel öffnen.

Suche

Nach Flaschennummer, Lieferant oder Kältemittel filtern.

KI optional

Mit einem öffentlichen Server können Fotos automatisch ausgewertet werden.

Android-App herunterladen

Lege die fertige APK unter releases/flaschen-ki.apk ab. Danach funktioniert der Download-Button automatisch.

APK herunterladen

Android kann beim ersten Start nach der Erlaubnis zur Installation aus unbekannten Quellen fragen.

Datenschutz

Die Standalone-Version speichert Flaschendaten lokal auf dem Gerät. Die optionale KI-Funktion sendet Bilder nur an die in der App eingetragene Serveradresse.
